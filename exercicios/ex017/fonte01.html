Trabalhando com Fontes

As fontes são um elemento crucial no design de uma página web. Elas ajudam a transmitir a mensagem e a personalidade do conteúdo. Em CSS, podemos controlar as fontes usando a propriedade font-family.

Propriedade font-family

A propriedade font-family permite especificar uma lista de fontes para um elemento. O navegador irá usar a primeira fonte disponível na lista. Se nenhuma das fontes estiver disponível, ele usará uma fonte genérica.

        font-family: Arial, sans-serif;
